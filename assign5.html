Working with Margins, Padding, Alignment, and Floating

It is important to understand the four particular CSS properties individually before putting them together (margin and padding properties + align and float properties).

Margins

Style sheet margins enables the developer to add empty space around the outside of the rectangular area for an element on a web page. The following are style properties for setting margins:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • margin (sets the top, right, bottom, and left margins as a single property)
Margins can be specified as auto, meaning the browser sets the margin in specific lengths or percentages. If the margin is set in a percentage, it is calculated based on the size of the entire page, not the size of the element. When using shorthand for setting margins, padding, or borders, there are actually three approaches, based on the amount of values you use:
  • One value-the size of all the margins
  • Two values-The size of the top/bottom margins and the left/right margins in that order.
  • Four values-The size of the top, right, bottom, left margins in that order.
Margin specifications are incredibly useful for element placement, but use with caution when setting specifications.

Padding Elements

Padding is similar to margins in that it adds extra space to elements, but the difference is where the space is located. Margins are added spcae outside of the element. Padding adds space inside the rectangular area of the element. For instance, if you creat a style rule for an element that has a width of 50 pixels and height of 30 pixels, and then set the padding to 5 pixels, the remaining content area will be 40 pixels by 20 pixels. Because the padding of an element appears in the element's content area, it assumes the same style as the content of the element, including background color. You can specify the padding of a style rule using the padding properties which work like the margin properties. Like margins, you can set the padding of style rules using inidivdual padding properties or the single padding property. If you place the text < p > element and give that element a white background, you can see where the padding is in relation to the text.

Alignment

Content on a webpage doesn't always fill the entire rectangular area, so it is helpful to control the alignment of the content. Even if text within a rectangular extends to multiple lines, alignment still enters the picture because you might want the text left-justified, right-justified, or centered. There are two style properties that enable you to control the alignment of elements: text-align and vertical-align. text-align aligns an element horizontally within its bounding area, and it can be set to: left, right, center, or justify. The vertical-align property is similar to text-align except that it is used to align elements vertically. It specifies how an element is aligned with it's parent, or the current line of elements on the page. Current line refers to the vertical placement of elements that appear within the same parent element. Basically, inline elements. If several inline elements appear on the same line, you can set their vertical alignments the same to align them vertically. For instance, a row of images that appear one after the next - the vertical-align property enables you to align them vertically. These are common values for use with the vertical-align property:

  • top-Aligns the top of an element with the current line.
  • middle-Aligns the middle of an element with the middle of it's parent.
  • bottom-Aligns the bottom of an element with the current line.
  • text-top-Aligns the top of an element with the top of it's parent.
  • baseline-Aligns the baseline of an element with the baseline of it's parent.
  • text-bottom-Aligns the bottom of an element with the bottom of it's parent.

Float Property

The float property allows elements to be moved around in the design such that other elements can wrap around them. You will often use this property for images, but you can float all sorts of elements. Elements float horizontally not vertically, so there are only two values: right and left. When used, an element that floats will float as far right or left as the containing element allows it.

Positioning

Relative positioning is the default type of positioning used by HTML. Absolute Positioning enables you to set the exact position of HTML content on a page. Although absolute positioning gives you the freedom to spell out exactly where an element is to appear, the position is still relative to any parent elements that appear on the page. Basically, absolute positioning enables you to specify the exact location of an element's rectangular area with respect to its parent area, which is very different from relative positioning. With the freedom of placing elements anywhere, you can run into some overlap. In this case CSS relies on the z-index of each element to determine which elemente is on the top and which is on the bottom.
